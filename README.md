## Part2 【win32程序设计实践】

### [实验一 Graphic](https://github.com/zyzisyz/Programming-Practice/tree/master/programing_code/graphic)

#### 要求

1. 建立一个自己的绘图程序，支持下列菜单功能：
    - 文件
        - 参数设置
        - 可设计成对话框形式
        - 退出
    - 绘图
        - 简单图形
            - 点和线
            - 三角形
            - 抛物线
            - 正弦曲线
        - 时钟
        - 复杂图形

            比如：弹球，随机数分布, 。。。。。。
2. 撰写实验报告：开发思路，主要技术，相关算法，主要编程点（书上的尽量少写，主要写自己的东西）
